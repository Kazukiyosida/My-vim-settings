## VIM

Vimの設定ファイル
 Windouws用

・Windowsでプラグインのインストール、実行に失敗するとき

  システムのvimrcに
	let $HOME = "C:/VIM/"
  を追記することで解決。

・runpde.BATについて
:Quickrun用
PROCESSINGの.pdeファイルをコンパイルし、C:\libraly\processingに出力、実行するものです。processing-java.exeにパスが通ってないと失敗します。
